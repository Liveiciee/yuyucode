# Fitur YuyuCode

1. **Pengelolaan Proyek Berbasis Git**
   - Otomatisasi branch per tugas
   - Isolasi lingkungan kerja dengan worktrees

2. **Eksekusi Perintah Terbatas (Agent Mode)**
   - Hanya menjalankan aksi yang didefinisikan (seperti write_file)
   - Mode latar belakang untuk tugas asinkron

3. **Sistem Action Block**
   - Format JSON untuk instruksi terstruktur
   - Dukungan multi-action dalam satu pesan

4. **Manajemen Konteks dan Working Directory**
   - Penentuan lokasi kerja spesifik per agen
   - Dukungan multi-agent dengan isolasi penuh

5. **Integrasi dengan Lingkungan Termux**
   - Kompatibel dengan Android melalui Termux
   - Akses ke sistem file dan tools CLI

6. **Logging dan Audit Trail**
   - Semua perubahan dicatat dan dikomit ke repositori
   - Mendukung pelacakan perubahan jangka panjang

7. **Ekspansi Fitur melalui Plugin (Rencana)**
   - Plugin untuk bahasa pemrograman tambahan
   - Integrasi dengan AI lokal (LLM offline)
